Hinweis: Die Datei Demo.txt enthält einige Zeilen Text.

Der Zugriffsmodus ist ein String. Es fehlen also die Hochkommata. Durch

das Öffnen mit "w+" wird die Datei aber auf die Länge 0 gekürzt, das heißt,

fgets() liest nichts und der Return-Wert ist NULL. Der richtige fopen-

Aufruf lautet also:

 

FILE *fp = fopen("Demo.txt", "r+");
